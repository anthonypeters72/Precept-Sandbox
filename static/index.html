📖 How to use Precept Sandbox

🔎 Search scripture

  • Genesis 1:1 → single verse
  • Genesis 1 → full chapter
  • Genesis 1:1-5 → verse range

🧠 Search by topic

  • judgment
  • faith hope love

⚡ Strong’s numbers

  • Add --strong to your query
  • Strong’s numbers are available for Old Testament verses only (for now)
  • Example: Genesis 1:1 --strong

🧭 Navigation dropdown

  • Select corpus → book → chapter → verse
  • Leave verse blank to load full chapter
Tip: add --strong for Strong’s output.
